# ElectronPM
ElectronPM is an universal package manager created to be able to use (almost) every package across all Linuxâ„¢ Distributions.

# How to use?
The available commands to be used in Electron will be;
```
electronpm install <file>

electronpm remove <package>

electronpm list

electronpm --help

electronpm --version
```
# How to install?
You are able to install ElectronPM on any distro via running this command in your terminal:
```
git clone https://github.com/Kenraaliskuutteri/ElectronPM.git
```
More options will be soon available.
